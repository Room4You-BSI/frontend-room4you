Sobre o projeto

O Room4You é uma aplicação web que visa facilitar o oferecimento e procura de quartos para usuários que buscam um lugar para se hospedar. Nele, o anunciante pode inserir informações do seu quarto, o qual deseja alugar, como preço, endereço e localização. Por outro lado, o usuário que deseja procurar um quarto, tem acesso a todas os anúncios de diferentes usuários, tendo assim, uma infinidade de opções para alocação, possibilitando, portanto, uma oferta e procura de quartos para moradia de forma eficiente e simplificada.
